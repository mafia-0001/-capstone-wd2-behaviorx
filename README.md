# 🌍 BehaviorX  
### Human Behavior Impact Analyzer

BehaviorX is a modern web application that analyzes daily user behavior and visualizes its impact on environmental and productivity factors through an interactive dashboard.

---

## 🚀 Live Demo
👉 https://behaviorx.vercel.app

---

## 📌 Problem Statement
People are often unaware of how their daily habits affect the environment and productivity. This project provides a system that analyzes user behavior and visually represents its impact to promote better decision-making.

---

## 🎯 Project Objective
- Analyze user behavior (mood, travel, screen time)
- Calculate environmental and productivity impact
- Provide interactive visualization through charts
- Increase awareness of sustainable lifestyle choices

---

## ✨ Key Features

- 🔐 Authentication System (Login / Signup)
- 🧠 Behavior Simulation Engine
- 📊 Interactive Dashboard (Charts & Metrics)
- 🌐 Live API Integration (Weather Data)
- ⚡ Redux State Management
- ✨ Smooth UI Animations
- 📱 Responsive Design
- 🚀 Deployed on Vercel

---

## 🛠️ Tech Stack

| Category | Technology |
|--------|-----------|
| Frontend | React (Vite) |
| State Management | Redux Toolkit |
| Styling | Tailwind CSS |
| Routing | React Router |
| API | Open-Meteo API |
| Charts | Recharts |
| Animation | Framer Motion |
| Deployment | Vercel |

---

## 🧩 System Architecture

User → React UI → Redux Store → Processing Logic → Dashboard → API

---

## 🔄 Workflow

Login → Simulator → Input Behavior → Process Data → Dashboard → Insights

---

## 🔐 Environment Variables

This project uses environment variables for API configuration:
VITE_API_URL=https://api.open-meteo.com/v1/forecast
> These variables are not stored in the repository and are configured securely in deployment.

## ⚠️ Limitations

- Uses simulated logic (not fully real-world accurate)
- No backend database (localStorage used)
- Limited behavioral parameters
